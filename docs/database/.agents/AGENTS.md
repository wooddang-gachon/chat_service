# Antigravity AI 작동 규칙

이 프로젝트에서 Antigravity AI가 작업을 수행할 때는 아래의 권한 제어 및 행동 규칙을 엄격하게 준수해야 합니다.

## 🔍 조회 및 읽기 권한
- 파일 시스템 조회(`list_dir`), 파일 내용 읽기(`view_file`), 코드 검색(`grep_search`) 등 단순 읽기 기능은 승인 없이 자유롭게 사용합니다.

## ✍️ 수정 및 실행 제한 규칙
- **선 동의 필수:** 파일 작성(`write_to_file`), 내용 수정(`replace_file_content`, `multi_replace_file_content`), 혹은 명령어 실행(`run_command`) 작업을 진행하기 전에, 반드시 사용자에게 변경할 코드의 내용이나 실행할 계획을 채팅으로 먼저 제안해야 합니다.
- **도구 호출 제한:** 사용자가 채팅을 통해 명시적으로 동의(예: "수정해 줘", "진행해 줘" 등)하기 전에는 수정/실행 관련 도구를 임의로 먼저 호출하지 않습니다.
- **예외 사항:** 사용자가 처음 요청할 때부터 명시적으로 수정을 지시한 경우에는 제안 단계 없이 바로 수정 툴을 사용하고 승인(Proceed) 팝업을 요청할 수 있습니다.
